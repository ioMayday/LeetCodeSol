# 刷题时常见错误
 - 数组下标越界，下标操作后再索引时，一定要保证在有效范围内，防止越界或为负
 - 双指针时，经常误用 left - right 作为长度, 最好使用 end - start 命名更清晰
 - 参考别人代码实现时，重要的是先搞懂思路，弄清每步操作到底是在干啥，自己再用代码去复现，实现更高效
